Waypoint

Waypoint hanya perlu edit koordinat (x,y) saja. Sesuai koordinat peta.


Terminal

Penjelasan terminal:
  • Type: 1 untuk stop, 2 atau 32 sama saja untuk jalan belok atau lurus
  • Id: Wajib beda, untuk membedakan terminal nama dan mempermudah mengubah terminal
  • X: Koordinat terminal sesuai koordinat peta
  • Y: Koordinat terminal sesuai koordinat peta
  • Theta: Arah jalan mobil sesuai koordinat peta
  • Max vx: Kecepatan robot saat didalam dan setelah melewati terminal
  • Max vy: (Abaikan)
  • Max vtheta: (Abaikan)
  • Radius area: radius area terminal
  • Lookahead distance: scan target untuk sistem kontrol movement mobil (lihat pada gambar)
  • Obs_scan_r: (lihat pada gambar)
  • Icp_score_max: Representasi nilai posisi salah atau ada perbedaan di peta nya. Semakin kecil nilainya semakin posisi mobil benar dan sedikit perbedaan di peta
  • Scan_min_x: (lihat pada gambar) (Abaikan)
  • Scan_max_x: (lihat pada gambar) (Abaikan)
  • Scan_min_y: (lihat pada gambar)
  • Scan_max_y: (lihat pada gambar)
  • Obs_threshold: (Abaikan)
